# 链表翻转
```c
node *head;

node *h=head;
node *hn=head->next;
node *hnn=hn->next;

head->next =NULL;

while(hn != NULL){
    hn->next =h;
    h= hn;
    hn = hnn;
    hnn = hnn->next;
}
```
